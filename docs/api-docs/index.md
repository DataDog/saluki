# API documentation

> [!NOTE]
> This is a placeholder page. In production, this page is replaced by a static build of the Rust API
> documentation generated by `cargo doc`. See `.github/workflows/docs.yml` for details.
>
> To generate API documentation locally, run:
>
> ```shell
> make generate-api-docs
> ```
>
> The output is written to `target/doc/`.
