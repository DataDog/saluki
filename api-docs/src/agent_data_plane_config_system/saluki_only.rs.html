1//! [`SalukiOnly`]: the parsed source values absent from the Datadog Agent schema, plus [`seed`].
2//!
3//! # What this is
4//!
5//! Saluki-only keys are values ADP consumes that the Datadog Agent schema does not publish, so they
6//! cannot be witnessed and driven like Datadog keys. Today they arrive through the same source as
7//! Datadog config (a `datadog.yaml`-shaped map / `DD_*` env), so [`SalukiOnly`] is the ADP-only
8//! sibling of the generated Datadog source model: a hand-written `Deserialize` view over that same
9//! merged map. [`seed`] then copies each present value into its home in [`SalukiConfiguration`].
10//! `seed` and the Datadog `drive` write disjoint fields, so they never contend.
11//!
12//! # Ownership doctrine
13//!
14//! A key's home is decided solely by membership in the vendored Datadog schema. The key name and
15//! prefix are irrelevant: `data_plane.*`, `dogstatsd_*`, and every other namespace can contain
16//! either kind of key.
17//!
18//! - A key present in the Datadog schema is witnessed by `DatadogConfiguration` and driven into
19//!   the model. It must not also have a `SalukiOnly` field, serde alias, or `seed` path.
20//! - A key absent from the Datadog schema is seeded here and copied into the model by [`seed`].
21//!
22//! A second SalukiOnly spelling for a Datadog key is not a compatibility feature. It is a duplicate
23//! source of truth and a bug; remove it. Removing that duplicate cannot affect customers because
24//! the Datadog-schema path remains authoritative.
25//!
26//! A dotted SalukiOnly key is reached from the environment by an algorithmic transform, never a
27//! lookup table or alias: `foo.bar` is reached by `DD_FOO_BAR`. For example,
28//! `data_plane.standalone_mode` is reached by `DD_DATA_PLANE_STANDALONE_MODE` and by no other name.
29//!
30//! The canonical inventory of these keys (with exact `yaml_path`, type, and default) is
31//! `SALUKI_KEYS` in the `datadog-agent-config-overlay-model` crate. Keep this struct in sync with
32//! it.
33//!
34//! # The one invariant (read before adding or debugging a value)
35//!
36//! This struct mirrors the source key hierarchy exactly. Each field is populated by plain
37//! serde from the merged map, so its path here must equal the real config-key path, with no
38//! `rename` and (aside from a documented multi-key `alias`) nothing papering over a mismatch:
39//!
40//! - a top-level key (`dogstatsd_tcp_port`, `aggregate_window_duration_seconds`) is a top-level
41//!   field of the same name;
42//! - a nested key (`apm_config.default_env`, `data_plane.metrics.v3.series.enabled`) is a field on
43//!   a matching chain of nested sub-structs.
44//!
45//! A mismatch does not error. The field deserializes to `None`, `seed` skips it, and the model
46//! keeps its default: the value silently fails to transport.
47//!
48//! # A component isn't getting a Saluki-only value
49//!
50//! 1. Value doesn't arrive at all: shape/path bug here, or a missing `seed` line. Confirm a field
51//!    exists whose full path equals the `yaml_path` in `SALUKI_KEYS` (nesting included), that its
52//!    type coerces the source form (durations are [`DurationString`], not `Duration`), and that
53//!    `seed` copies it to the model destination the component reads.
54//! 2. Value arrives wrong only when the key is absent: the default bug is not here. The default
55//!    lives in the model `Default` in `agent-data-plane-config`, because these fields are
56//!    `Option<T>` and `seed` writes only when set. Fix the model `Default`, not this struct.
57//! 3. Then add or extend the round-trip test below (set the real key, assert the model field). A
58//!    missing test is why a silent transport failure was not caught.
59//!
60//! # Adding a Saluki-only value
61//!
62//! Add the field (path == `yaml_path`), pick a coercing type, add the `seed` line to the model
63//! home, set the model `Default`, and add a round-trip test.
64//!
65//! [`seed`]: SalukiOnly::seed

79/// The parsed Saluki-schema-only configuration, shaped to mirror the source key hierarchy.
80///
81/// Flat keys are top-level fields; nested keys live on matching nested sub-structs. Every field is
82/// `#[serde(default)]` and `Option`-typed (or an empty collection), so a deployment may set no
83/// Saluki-only keys at all and each falls back to its model default. Deserialized from the same
84/// merged map as the Datadog source model, so unknown (Datadog) keys are ignored.
85#[derive(Clone, Debug, Default, Deserialize)]
86#[serde(default)]
87pub struct SalukiOnly {
88    // ── top-level scalar keys ────────────────────────────────────────────────
89    /// Internal-telemetry verbosity (`metrics_level`).
90    pub metrics_level: Option<String>,
91    /// Remote-agent IPC string interner byte budget (`remote_agent_string_interner_size_bytes`).
92    pub remote_agent_string_interner_size_bytes: Option<usize>,
93    /// Checks IPC endpoint (`checks_ipc_endpoint`).
94    pub checks_ipc_endpoint: Option<String>,
95    /// Process memory limit (`memory_limit`), given as a bare integer number of bytes or a
96    /// byte-size string such as `512MB`. `ByteSize` accepts both forms, so a numeric value does not
97    /// fail the load.
98    pub memory_limit: Option<ByteSize>,
99    /// Memory-accounting slop fraction (`memory_slop_factor`).
100    pub memory_slop_factor: Option<f64>,
101    /// Encoder flush timeout, in seconds (`flush_timeout_secs`).
102    pub flush_timeout_secs: Option<u64>,
103    /// Maximum metrics per payload (`serializer_max_metrics_per_payload`).
104    pub serializer_max_metrics_per_payload: Option<usize>,
105
106    // ── DogStatsD listener/context/mapper keys (all top-level) ────────────────
107    /// TCP listen port (`dogstatsd_tcp_port`).
108    pub dogstatsd_tcp_port: Option<u16>,
109    /// Baseline number of receive buffers (`dogstatsd_buffer_count`).
110    pub dogstatsd_buffer_count: Option<usize>,
111    /// Maximum number of receive buffers (`dogstatsd_buffer_count_max`).
112    pub dogstatsd_buffer_count_max: Option<usize>,
113    /// Whether to bind multiple UDP sockets via `SO_REUSEPORT` (`dogstatsd_autoscale_udp_listeners`).
114    pub dogstatsd_autoscale_udp_listeners: Option<bool>,
115    /// Whether to relax decoder strictness (`dogstatsd_permissive_decoding`).
116    pub dogstatsd_permissive_decoding: Option<bool>,
117    /// Maximum cached metric contexts (`dogstatsd_cached_contexts_limit`).
118    pub dogstatsd_cached_contexts_limit: Option<usize>,
119    /// Maximum cached tagsets (`dogstatsd_cached_tagsets_limit`).
120    pub dogstatsd_cached_tagsets_limit: Option<usize>,
121    /// Explicit byte budget for the context interner (`dogstatsd_string_interner_size_bytes`).
122    pub dogstatsd_string_interner_size_bytes: Option<u64>,
123    /// Whether to allow heap allocations for contexts (`dogstatsd_allow_context_heap_allocs`).
124    pub dogstatsd_allow_context_heap_allocs: Option<bool>,
125    /// Floor for metric sample rates (`dogstatsd_minimum_sample_rate`).
126    pub dogstatsd_minimum_sample_rate: Option<f64>,
127    /// Mapper string interner entry count (`dogstatsd_mapper_string_interner_size`).
128    pub dogstatsd_mapper_string_interner_size: Option<u64>,
129
130    // ── aggregation keys (all top-level) ──────────────────────────────────────
131    /// Aggregation window size, in seconds (`aggregate_window_duration_seconds`).
132    pub aggregate_window_duration_seconds: Option<u64>,
133    /// Maximum contexts per aggregation window (`aggregate_context_limit`).
134    pub aggregate_context_limit: Option<usize>,
135    /// Aggregator flush period (`aggregate_flush_interval`).
136    pub aggregate_flush_interval: Option<DurationString>,
137    /// Whether open aggregation windows are flushed (`aggregate_flush_open_windows`).
138    pub aggregate_flush_open_windows: Option<bool>,
139    /// Passthrough idle flush delay (`aggregate_passthrough_idle_flush_timeout`).
140    pub aggregate_passthrough_idle_flush_timeout: Option<DurationString>,
141
142    // ── OTLP metric context keys (all top-level) ──────────────────────────────
143    /// Whether to allow heap allocations for OTLP contexts (`otlp_allow_context_heap_allocs`).
144    pub otlp_allow_context_heap_allocs: Option<bool>,
145    /// Maximum cached OTLP metric contexts (`otlp_cached_contexts_limit`).
146    pub otlp_cached_contexts_limit: Option<usize>,
147    /// Maximum cached OTLP tagsets (`otlp_cached_tagsets_limit`).
148    pub otlp_cached_tagsets_limit: Option<usize>,
149    /// OTLP context interner byte budget (`otlp_string_interner_size`). Accepts human-readable sizes
150    /// such as `2MiB` as well as a plain byte count.
151    pub otlp_string_interner_size: Option<ByteSize>,
152
153    // ── nested sections ───────────────────────────────────────────────────────
154    /// Cross-cutting data-plane knobs (`data_plane.*`).
155    pub data_plane: DataPlane,
156    /// APM trace knobs (`apm_config.*`).
157    pub apm_config: ApmConfig,
158    /// OTLP receiver and trace knobs (`otlp_config.*`).
159    pub otlp_config: OtlpConfig,
160    /// OTTL span-drop filter (`ottl_filter_config`).
161    pub ottl_filter_config: Option<OttlFilterConfig>,
162    /// OTTL span-transform processor (`ottl_transform_config`).
163    pub ottl_transform_config: Option<OttlTransformConfig>,
164}

369/// Parses an OTTL error mode string, defaulting to the model default (`Propagate`) when absent or
370/// unrecognized.
371fn parse_ottl_error_mode(mode: Option<String>) -> OttlErrorMode {
372    match mode.as_deref() {
373        Some("ignore") => OttlErrorMode::Ignore,
374        Some("silent") => OttlErrorMode::Silent,
375        _ => OttlErrorMode::Propagate,
376    }

379impl SalukiOnly {
380    /// Copies each present Saluki-only value into its destination in `config`.
381    ///
382    /// Absent values leave the field at its model default. The Datadog `drive` writes a disjoint set
383    /// of fields, so it does not matter whether `seed` runs before or after the drive.
384    pub(crate) fn seed(&self, config: &mut SalukiConfiguration) {
385        // control
386        if let Some(v) = self.data_plane.stop_timeout {
387            config.control.stop_timeout = v;
388        }
389        if let Some(v) = self.data_plane.standalone_mode {
390            config.control.standalone_mode = v;
391        }
392        if let Some(v) = self.data_plane.checks.enabled {
393            config.control.checks = v;
394        }
395        if let Some(v) = self.memory_limit {
396            config.control.memory_limit = v.as_u64();
397        }
398        if let Some(v) = self.memory_slop_factor {
399            config.control.memory_slop_factor = v;
400        }
401        if let Some(v) = self.remote_agent_string_interner_size_bytes {
402            config.control.ipc.remote_agent_string_interner_size_bytes = v;
403        }
404
405        // shared
406        if let Some(v) = self.metrics_level.clone() {
407            config.shared.metrics_level = v;
408        }
409        if let Some(v) = self.flush_timeout_secs {
410            config.shared.metrics_encoding.flush_timeout = Duration::from_secs(v);
411        }
412        if let Some(v) = self.serializer_max_metrics_per_payload {
413            config.shared.metrics_encoding.max_metrics_per_payload = v;
414        }
415        if let Some(v) = self.data_plane.metrics.v3.series.enabled {
416            config.shared.metrics_encoding.v3_series_enabled = v;
417        }
418
419        // domains.dogstatsd
420        let dsd = &mut config.domains.dogstatsd;
421        if let Some(v) = self.dogstatsd_tcp_port {
422            dsd.listeners.tcp_port = v;
423        }
424        if let Some(v) = self.dogstatsd_buffer_count {
425            dsd.listeners.buffer_count = v;
426        }
427        if let Some(v) = self.dogstatsd_buffer_count_max {
428            dsd.listeners.buffer_count_max = v;
429        }
430        if let Some(v) = self.dogstatsd_autoscale_udp_listeners {
431            dsd.listeners.autoscale_udp_listeners = v;
432        }
433        if let Some(v) = self.dogstatsd_permissive_decoding {
434            dsd.listeners.permissive_decoding = v;
435        }
436        if let Some(v) = self.dogstatsd_cached_contexts_limit {
437            dsd.contexts.cached_contexts_limit = v;
438        }
439        if let Some(v) = self.dogstatsd_cached_tagsets_limit {
440            dsd.contexts.cached_tagsets_limit = v;
441        }
442        if let Some(v) = self.dogstatsd_string_interner_size_bytes {
443            dsd.contexts.string_interner_size_bytes = Some(v);
444        }
445        if let Some(v) = self.dogstatsd_allow_context_heap_allocs {
446            dsd.contexts.allow_context_heap_allocs = v;
447        }
448        if let Some(v) = self.dogstatsd_minimum_sample_rate {
449            dsd.contexts.minimum_sample_rate = v;
450        }
451        if let Some(v) = self.dogstatsd_mapper_string_interner_size {
452            dsd.mapper.string_interner_size = v;
453        }
454        if let Some(v) = self.aggregate_window_duration_seconds {
455            dsd.aggregation.window_duration_seconds = v;
456        }
457        if let Some(v) = self.aggregate_context_limit {
458            dsd.aggregation.context_limit = v;
459        }
460        if let Some(v) = self.aggregate_flush_interval {
461            dsd.aggregation.flush_interval = v.as_duration();
462        }
463        if let Some(v) = self.aggregate_flush_open_windows {
464            dsd.aggregation.flush_open_windows = v;
465        }
466        if let Some(v) = self.aggregate_passthrough_idle_flush_timeout {
467            dsd.aggregation.passthrough_idle_flush_timeout = v.as_duration();
468        }
469
470        // domains.otlp
471        let otlp = &mut config.domains.otlp;
472        if let Some(v) = self.otlp_allow_context_heap_allocs {
473            otlp.contexts.allow_context_heap_allocs = v;
474        }
475        if let Some(v) = self.otlp_cached_contexts_limit {
476            otlp.contexts.cached_contexts_limit = v;
477        }
478        if let Some(v) = self.otlp_cached_tagsets_limit {
479            otlp.contexts.cached_tagsets_limit = v;
480        }
481        if let Some(v) = self.otlp_string_interner_size {
482            otlp.contexts.string_interner_size = v.as_u64();
483        }
484        if let Some(v) = self.otlp_config.receiver.protocols.http.transport.clone() {
485            otlp.receiver.http.transport = v;
486        }
487        if let Some(v) = self.otlp_config.traces.string_interner_size {
488            otlp.traces.string_interner_size = v.as_u64();
489        }
490        if let Some(v) = self.otlp_config.traces.enable_otlp_compute_top_level_by_span_kind {
491            otlp.traces.enable_compute_top_level_by_span_kind = v;
492        }
493        if let Some(v) = self.otlp_config.traces.ignore_missing_datadog_fields {
494            otlp.traces.ignore_missing_datadog_fields = v;
495        }
496        // Replace the schema-translated endpoints so ADP cannot inherit the Core Agent's ports.
497        //
498        // TODO(#2177): Remove this override when ADP uses the canonical endpoint keys and defaults.
499        otlp.receiver.grpc.endpoint = self
500            .data_plane
501            .otlp
502            .receiver_grpc_endpoint_temporary
503            .clone()
