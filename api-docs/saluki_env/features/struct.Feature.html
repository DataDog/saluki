Struct Feature

pub struct Feature { /* private fields */ }

Features are distinct markers or indicators of a particular technology or platform that is present.

In general, features represent the type of environment that the application is running in, such as the Kubernetes feature being present indicating that the application is running in Kubernetes, and so on.
