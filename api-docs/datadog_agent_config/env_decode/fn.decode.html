pub fn decode(raw: &str, how: EnvDecode) -> Result<Value, String>

Decodes raw into the JSON value how prescribes.

§Errors

Returns a human-readable message when raw is malformed for how (invalid JSON, a bad boolean or number, or a traces_span token that is not name=rate). The caller pairs it with the environment variable name.
