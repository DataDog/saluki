pub fn version_string() -> Option<&'static str>
Expand description

Raw Agent version string that ADP was built against, as detected at build time.

This is the verbatim DD_AGENT_VERSION value (for example, "7.81.0-full" or "nightly"), suitable for display in diagnostics. Returns None when DD_AGENT_VERSION was not set at build time.
