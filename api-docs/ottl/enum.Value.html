Enum Value 

Source 
pub enum Value {
    Nil,
    Bool(bool),
    Int(i64),
    Float(f64),
    String(Arc<str>),
    Bytes(Arc<[u8]>),
    List(Vec<Value>),
    Map(HashMap<String, Value>),
}
Expand description

Represents all possible values in OTTL expressions and function arguments. Uses Arc for strings and Arc<u8> for bytes to enable cheap cloning.
