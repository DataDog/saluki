Expand description

A dense store using contiguous array storage.

This store grows unbounded to accommodate any range of indices. Itโs memory-efficient when the indices are clustered together, but can use significant memory if indices are widely scattered.

Use this store when:

  • You have a bounded range of input values
  • Memory usage is not a concern
  • You need the fastest possible insertion performance
