Adds an exposed port to the container.

Exposed ports are ports mapped from inside the container to an ephemeral port on the host. The protocol should be either tcp or udp. A port on the host side is picked from the โlocalโ port range. For example, on Linux the range is defined by /proc/sys/net/ipv4/ip_local_port_range.

When starting the driver via Driver::start, the ephemeral port mappings will be returned in DriverDetails.
