Enum Connection 

Source 
pub enum Connection {
    Tcp(TcpStream, SocketAddr),
    Unix(UnixStream),
}
Expand description

A connection-oriented socket.

This type wraps network sockets that operate in a connection-oriented manner, such as TCP or Unix domain sockets in stream mode.

Variants§

§

Tcp(TcpStream, SocketAddr)

A TCP socket.

§

Unix(UnixStream)
