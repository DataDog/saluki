Agent Data Plane

Plug-and-play

Saluki's component model makes it easy to create and connect components together to build full telemetry data processing pipelines.

Efficient and deterministic

Saluki is architected to be incredibly resource efficient and deterministic: predict and limit your resource usage ahead of time.

Fit for purpose

Saluki powers Agent Data Plane, the experimental, next-generation data plane for the Datadog Agent: big or small, Saluki should handle it all.
